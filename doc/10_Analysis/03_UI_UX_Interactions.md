# 03. UI/UX Strategy & Micro-Interactions

리트젠의 첨단 광학 기술을 직관적으로 이해할 수 있는 시각적 경험을 설계합니다.

## 1. 핵심 마이크로 인터랙션 (Micro-Interactions)
Framer Motion을 활용하여 빛의 물성을 역동적으로 시각화합니다.

### 1.1 UV/LED & UV SYSTEM (정밀 조사 및 고속 경화)
- **효과**: 사용자가 호버(Hover) 시, Electric Blue/Violet 색상의 빛이 정밀하게 쏘아지는 애니메이션.
- **매칭**: 빛이 닿는 순간 하단의 액체 그래픽(접착제/잉크/코팅)이 즉각적으로 단단하게 굳어지는 파티클 효과 연출.

### 1.2 IR SYSTEM (소재 내부 균일 가열)
- **효과**: 3D 다층 소재 그래픽을 스크롤할 때, 적외선이 소재 표면을 통과해 깊숙이 스며드는 모션.
- **매칭**: 빛이 침투한 후 소재 내부 전체가 균일하게 발광(Glow)하며 내부 가열 원리를 시각화.

### 1.3 방열 시스템 (Heat Dissipation System)
- **효과**: 붉은색 열기 파티클이 생성되다가, 시스템 작동 시 푸른색 기류가 열기를 외부로 배출하는 애니메이션.
- **매칭**: 장비의 안정성과 수명 연장 효과를 기술적으로 강조.

### 1.4 살균 프로세스 (Sterilization Process)
- **효과**: 세균/바이러스 형태의 미세 입자가 부유하다가 보라색(Violet) 파장의 빛이 지나가면 파괴되거나 사라지는 효과.

## 2. 기술 사양 비교 그리드 (Tech Spec Grid)
엔지니어와 구매 담당자를 위한 핵심 UI 컴포넌트입니다.

- **디자인**: Deep Navy 배경에 하이라이트 수치(Electric Blue/Violet) 강조.
- **데이터 연동**: Firestore/Sheets를 통해 실시간 데이터 바인딩.
- **주요 비교 항목**:
  - 조사 형태 (Form Factor): 점(Spot), 선(Linear), 면(Area), 컨베이어
  - 작동 원리 (Mechanism): 자외선 vs 적외선
  - 타깃 소재: 접착제, 잉크, 코팅 등
  - 실제 추가 항목: 최대 조사 강도, 파장대(nm), 냉각 방식 등

## 3. 산업 구별 페이지 (Applications)
- **레이아웃**: 비주얼(3D 모델) ➔ 공정 설명(AI 마케팅 문구) ➔ 추천 장비 매칭의 3단 그리드 구조로 전개.
- **내비게이션**: 상단 고정(Sticky) 탭을 통한 빠른 산업군 이동 가능.
