# Project: Lichtzen Home Ver 2.0

리트젠 글로벌 웹사이트 리뉴얼 프로젝트. "The Precision of Light"라는 브랜드 정체성을 현대적인 웹 기술로 구현합니다.

## 🎯 목표 (Objectives)
- **글로벌 대응**: 영문/중문을 포함한 완벽한 다국어 서비스 및 해외 바이어 유입 극대화.
- **기술적 압도성**: UV/LED/MIR 기술을 마이크로 인터랙션과 고화질 데이터 시각화로 전달.
- **비즈니스 리드 수집**: CRM 시스템(Inquiry Form) 연동을 통한 잠재 고객 확보.

## 🛠️ 기술 스택 (Tech Stack)
- **Frontend**: Next.js 14+ (App Router), React 18
- **Styling**: Tailwind CSS v4, Vanilla CSS
- **Animation**: Framer Motion (빛 파동 및 입자 인터랙션), Anime.js
- **Database**: Firebase (Firestore) - 제품 마케팅 시트 및 고객 문의 연동
- **CMS**: Google AI Studio (Gemini) 연동 다국어 콘텐츠 관리
- **Deployment**: Vercel (CI/CD)

## 🎨 디자인 시스템 (Design System)
- **Primary Color**: Deep Navy (#0A0E1A) - 고요함과 우주의 깊이
- **Accent Color**: Electric Violet (#8B5CF6) - UV 기술의 상징
- **Accent Color 2**: Cyan Blue (#06B6D4) - 미래지향적 정밀함
- **Typography**: Inter (Global), Outfit (Headers)

## 📁 주요 경로 (Key Paths)
- `.planning/`: 프로젝트 로드맵 및 상태 관리 (GSD 표준)
- `doc/10_Analysis/`: 기혹 및 요구사항 분석 아카이브
- `doc/20_Design/`: 메뉴별 상세 콘텐츠 디자인 가이드
- `src/`: Next.js 소스 코드 (예정)
